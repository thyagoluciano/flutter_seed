Introdução

Projeto desenvolvido para estudo de desenvolvimento de aplicativos com Flutter, criei essa documentação para facilitar a configuração de um novo projeto utilizando Firebase e configurado por Flavor os ambientes de (DEV, PROD)

O que foi feito ate agora?

Features

  • Configuração do projeto usando Flavor para DEV e PROD
  • Adicionado integração com firebae com arquivos especificos para DEV e PROD
  • Firebase Login com Email/Password
  • Firebase Login com Google
